HTML Table Reference

Common HTML Tags

Tag Name Description
<h1> and </h1> Heading Used for main headings
<p> and </p> Paragraph Used for paragraphs
<a> and </a> Anchor Used to specify the text that is the anchor/placeholder for a link
<button> and </button> Button Used to display buttons
<link> Link Used to link HTML and CSS sheets
<table> and </table> Table Used to hold information in a tabular format
<title> and </title> Title Tells browser what the page contains

Common HTML Attributes

Attribute Name Description
title Title Provides additional context to the anchor text
href Href Added at the opening tag of anchor element to provide the address or path where a file is located

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span
rowspan Row Span Defines how many rows a td element should span
